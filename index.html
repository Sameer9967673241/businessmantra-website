Family Visa Requirements

For HUSBAND & WIFE

  • Sponsor passport copy
  • Visa copy & Emirates ID copy
  • Ejari
  • Wife passport copy
  • Marriage certificate copy (attested)
  • Passport size picture (white bg)

For KIDS

  • Sponsor passport, Visa, EID copy
  • Kids passport copy
  • Ejari
  • Birth certificate copy (attested)
  • Passport size picture (white bg)
  • CNIC copy (only for Pakistanis)
